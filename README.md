# seminaire-labo
Support de présentation de mon séminaire de recherche (2026).

## Afficher le diaporama

    pixi init
    pixi shell
    voila --template=reveal slideshow.ipynb

## Modifier les diapos

    jupyter-lab

## Convertir les diapos en HTML

    jupyter nbconvert --to html --template reveal slideshow.ipynb
